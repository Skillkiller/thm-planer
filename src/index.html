Anleitung - Klicke zum Ausklappen
  • Navigiere zum THM Organizer
  • Bewege Maus über "VERANSTALTUNGEN"
  • Klicke "Stundenpläne" bei deinem jeweiligen Fachbereich
  • Klicke "Filter"
  • Wähle dein Studiengang aus
  • Klicke "Exportieren"
  • Klicke "Kalender-Abonnement"
  • Öffne den angezeigten Link in einem neuen Tab. Die Datei wird dann heruntergeladen
  • Gehe zurück zum THM-Planer
  • Wähle die heruntergeladene Datei aus
  • Wähle deine Module aus
  • Schaue welche Termine die Module haben
  • Wechsel die Ansichten
